# 📊 Vrinda Store Sales Dashboard

## 📌 About the Project

This is my first Excel Data Analytics project. It analyzes Vrinda Store's annual sales data using Microsoft Excel and presents key business insights through an interactive dashboard.

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## 🔍 Key Insights

- Women contributed more sales than men.
- Most orders were successfully delivered.
- Amazon was the top-performing sales channel.
- Maharashtra generated the highest sales.
- Adult women were the highest purchasing customer segment.
